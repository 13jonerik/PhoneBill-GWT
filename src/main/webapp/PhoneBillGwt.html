README

Phone Bill Application version 5.0 Developed by Jon-Erik Svenson for CS410J - Advanced Java Programming

This project is designed to familiarize us with Google Web Toolkit by creating a GUI in Java. A user can create a phone call validated by the program, which creates a phone bill for that user. The program keeps track of all existing phone bills and allows the user to search for calls that happened between certain dates. Other functionality includes sorting phone calls for the user, removing equal phone calls, printing the calls in a 'pretty' format, and providing helpful content.

Help

Need help using this GUI? You've come to the right place...

Adding a phone call:

Customer name - Any string of characters, as long as it's not empty
Caller number - The number that the call came from, in XXX-XXX-XXXX format, ex 645-432-2343
Callee number - The number that was called, in the same format
Start date - A valid date, in MM/DD/YYYY form, ex. 11/07/1990
End date - Also in MM/DD/YYYY form
Start time - A valid time in 12 hour format, formatted XX:XX, ex. 3:30 or 12:21
End time - also in 12 hour format, XX:XX
AM / PM - Use the toggle button to specify AM or PM

Adding a phone call will first validate that the inputs are correct, and then
store the information to a phone bill for you. If the customer name is new, a new
bill will be created. If you would like to see all of your bills, enter your name
in the prettify text box and your calls will be retrieved!

Searching a phone call:

** Arguments should be in the same format as specified above **

Provide a customer name, start and end times, and a window will pop up with
all calls found that fall within the time frame provided.

Prettify:

Need to see all of the calls in a phone bill? Simply enter a customer name!

Other information:

Calls that have the same customer name, caller number, and start time are considered
equal, and thus aren't added to the phone bill.
